# Social Media Channel — 8-Agent Pipeline

## Pipeline

```
Audience Discovery → Content Monitor → Analysis → Content Strategist → Content Creator → Publishing → Engagement → Performance Learning
                                                                                                                          ↓
                                                                                                                marketing_knowledge
```

## Agents

| # | Agent File | Role |
|---|-----------|------|
| 01 | `marketing-social-audience.md` | Find target segments, competitor accounts, communities |
| 02 | `marketing-social-monitor.md` | Track competitor posts, trending formats, viral content |
| 03 | `marketing-social-analysis.md` | Find winning formats, optimal times, effective hooks |
| 04 | `marketing-social-strategy.md` | Create content calendar with A/B tests |
| 05 | `marketing-social-creation.md` | Write platform-native posts, captions, scripts |
| 06 | `marketing-social-publishing.md` | Schedule and publish across platforms |
| 07 | `marketing-social-engagement.md` | Monitor and respond to comments, DMs, mentions |
| 08 | `marketing-social-performance.md` | Track engagement, reach, growth — extract insights |

## Run Modes

### Full Run (new niche)
Run agents 01 through 06 in sequence.

### Engagement (ongoing)
Run agent 07 regularly to monitor and respond.

### Performance Review (periodic)
Run 08 → 03 → 04 → 05.

### Content Only
Run 04 → 05 → 06 (Strategy → Creation → Publishing).

## Content Mix (80/20 Rule)

**80% Value Content:**
- Educational (30%) — teach your audience something useful
- Social Proof (20%) — show real results and testimonials
- Behind the Scenes (10%) — humanize the brand
- Engagement Bait (10%) — drive comments and shares
- Community (5%) — engage with audience questions
- Trend Rides (5%) — adapt trending formats to your niche

**20% Promotional:**
- Product features and announcements
- Customer success stories
- Limited offers and CTAs

## Key Principles

- Platform-native content always. Don't cross-post identical content.
- Every platform has different format preferences and audiences.
- Engagement is a conversation, not a billboard.
- The shared brain receives insights from social that inform ad copy and SEO content.
- Track everything — the Performance Learning agent closes the loop.
