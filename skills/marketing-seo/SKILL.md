# SEO Channel — 8-Agent Pipeline

## The 3-Bucket Strategy (from Ahrefs research)

For every query, ask: "Can AI fully satisfy the user?"
- **NO** → Traditional SEO (content, links, technical). These queries still get clicks.
- **YES** → Brand mention strategy. Get mentioned on pages AI pulls from.
- **BOTH** → Pursue both strategies for high-value queries.

When AI Overviews appear, even the #1 result loses ~58% of clicks. Don't chase rankings for those queries — chase mentions.

## Pipeline

```
Query Classifier → Discovery Monitor → Mention Hunter → Analysis → Content Strategist → Content Creator → Outreach → Performance Learning
                                                                                                                              ↓
                                                                                                                    marketing_knowledge
```

## Agents

| # | Agent File | Role |
|---|-----------|------|
| 01 | `marketing-seo-query-classifier.md` | Expand and classify keywords by AI satisfiability |
| 02 | `marketing-seo-discovery.md` | Track which domains AI assistants cite |
| 03 | `marketing-seo-mention-hunter.md` | Find specific pages to get mentioned on |
| 04 | `marketing-seo-analysis.md` | Find patterns in citations, competitor shifts |
| 05 | `marketing-seo-strategy.md` | Plan content briefs (blog, YouTube, Reddit, Quora) |
| 06 | `marketing-seo-creation.md` | Generate full content pieces from briefs |
| 07 | `marketing-seo-outreach.md` | Execute brand mention outreach campaigns |
| 08 | `marketing-seo-performance.md` | Track rankings, AI citations, extract insights |

## Run Modes

### Full Run (new niche)
Run agents 01 through 07 in sequence.

### Performance Review (periodic)
Run 08 → 04 → 05 (Performance feeds back into Analysis and Strategy).

### Content Only
Run 05 → 06 (Strategy → Creation from existing briefs).

### Outreach Only
Run 07 against existing published content.

### Learning Loop
Wait 2-4 weeks after content is published, then run 08 → 04 → 05.

## Key Principles

- Action queries (calculator, builder, generator) are gold — AI can't satisfy them.
- Brand mentions correlate more with AI visibility than backlinks.
- YouTube is disproportionately cited by Google AI Overviews.
- Reddit and Quora are trust signals for AI citation.
- Bottom-of-funnel content converts. Top-of-funnel is secondary.
- Every blog must link to money pages. No orphan content.
- FAQ sections capture AI Overview snippets.
