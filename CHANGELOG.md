# Changelog

## [1.0.0] — 2026-04-06

### Added
- 30 marketing agents across 3 channels (Meta Ads, SEO, Social Media)
- 6 Infrastructure agents (Tool Scanner, Strategy Monitor, Cost Optimizer, System Auditor, Upgrade Deployer, Reporting)
- Shared `marketing_knowledge` brain with Supabase schema
- 4 slash commands: `/marketing-setup`, `/marketing-run`, `/marketing-learn`, `/marketing-report`
- `install.sh` and `uninstall.sh` for one-command setup
- MIT license

### Channels
- **Meta Ads (8 agents)**: Discovery → Collection → Analysis → Strategy → Creation → Campaign → Performance Learning → Creative Intel
- **SEO (8 agents)**: Query Classifier → Discovery Monitor → Mention Hunter → Analysis → Content Strategist → Content Creator → Outreach → Performance Learning
- **Social Media (8 agents)**: Audience Discovery → Content Monitor → Analysis → Content Strategist → Content Creator → Publishing → Engagement → Performance Learning

### Key Features
- Cross-channel learning: insights from one channel improve all others
- Self-improving: Infrastructure agents scan for new tools and strategies weekly
- Works with existing tools: claude-seo, claude-ads, Playwright MCP, Adspirer MCP, Supabase
